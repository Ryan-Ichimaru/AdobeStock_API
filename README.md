# Stock Photo Metadata Automation System (v2)

Mac 上で動作する、「ストックフォト・メタデータ自動生成システム」です。
JPEG画像をフォルダに置くだけで、AI（GPT-4o-mini）がタイトルとキーワードを生成し、CSVに出力してファイルを整理します。

## セットアップ手順

### 1. 仮想環境の有効化と依存関係の確認
ターミナルを開き、本ディレクトリ (`Stock_Factory`) に移動して、以下のコマンドで仮想環境を有効化してください。
```bash
source .venv/bin/activate
```

### 2. OpenAI APIキーのセキュリティ登録 (初回のみ)
macOSのキーチェーンを使用して安全にAPIキーを管理します。
以下のコマンドを実行し、プロンプトに従って `OPENAI_API_KEY` を入力してください。
```bash
python set_api_key.py
```

### 3. システムの起動
以下のコマンドで、フォルダ監視システムを起動します。
```bash
python system_runner.py
```

## 期待される動作

1. **ディレクトリ監視**: 起動中、常に `1_Input` フォルダを監視します。
2. **JPEG解析**: **JPEGファイル (.jpg / .jpeg)** が投下されると、全自動で **GPT-4o-mini** が画像を解析します。
3. **メタデータ生成**: ストックフォトに最適な「タイトル」と「50個のキーワード」を生成します。
4. **CSV出力 & 整理**: 生成されたデータは `2_Output/adobe_stock_master.csv` に追記され、画像ファイル自体も `2_Output` フォルダに移動されます。

あとは、出力されたCSVと画像をストックフォトサイトにアップロードするだけです。

---
*注: 本バージョンでは、AIによる画像拡大（Upscale）および毎朝のトレンド分析機能は削除され、タグ付けと整理に特化しています。*
